# Pricing — RepoToPitch

RepoToPitch is open source. The source code is free, always.

If you want a pre-built Windows application that just works — no Rust compiler, no build environment, no setup beyond installing Ollama — that is what the paid editions are for.

## Free — Community Edition

**The source code. Build it yourself.**

- Full feature set — no artificial limitations
- All 10 synthesis artifacts included
- PDF, Markdown, and PPTX export
- Multi-repo support
- Requires: Node.js 18+, Rust (stable), WiX Toolset v3

→ [Clone the repo and follow the Developer Quick Start](README.md#quick-start-developer)
